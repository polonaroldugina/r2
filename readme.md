Бот-ассистент в ВК
Для запуска необходимо прописать токен и id сообщества вк в файле vk_server.py
Также запустить файл main

Бот обладает функциями ассистента для студента - может показвать расписание, погоду, а также советовать рецепты на завтрак. Также файлы хрнаят информацию о истории запросов пользователей, анализируя их, хранит расписание и списки учебных классов.

Бот сообщает об ошибках пользователю, также и о некоректном вводе данных.
